# Исправления ошибок компиляции WebRTC

## Исправленные ошибки

### 1. `Unresolved reference 'pendingIceCandidates'`
**Проблема:** Отсутствовало объявление переменной для хранения ожидающих ICE кандидатов
**Решение:** Добавили объявление:
```kotlin
// Очередь ожидающих ICE кандидатов
private val pendingIceCandidates = mutableListOf<IceCandidate>()
```

### 2. `Unresolved reference 'enableRtpDataChannel'`
**Проблема:** Свойство `enableRtpDataChannel` не существует в текущей версии WebRTC
**Решение:** Удалили несуществующее свойство и добавили комментарий:
```kotlin
// enableRtpDataChannel не существует в этой версии WebRTC
```

### 3. `Cannot infer type for this parameter`
**Проблема:** Компилятор не мог определить тип параметра в `forEach`
**Решение:** Явно указали тип:
```kotlin
pendingIceCandidates.forEach { candidate: IceCandidate ->
    // обработка кандидата
}
```
